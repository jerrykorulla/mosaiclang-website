About MOSAIC Language

MOSAIC Language is designed for efficient modelling and simulation in C++. It provides a robust framework for developers and researchers to create simulations with ease.

Features

  • Easy-to-use API for simulation development
  • Support for various simulation models
  • High performance and efficiency
  • Extensive documentation and community support
